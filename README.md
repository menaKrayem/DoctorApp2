الاسم منة الله وائل ابو كريم كلية فلسطين التقنية فرع دير البلح 
اسم المشروع Doctor App2
وصف التطبيق: تطبيق مصمم لتسهيل عملية حجز مواعيد الأطباء وتقديم نصائح صحية ويتميز التطبيق بواجهة مستخدم عصرية، وتكامل مع البيانات الحية (APIs)، وتخزين البيانات محلياً. 
نظام تسجيل دخول وإنشاء حساب آمن باستخدام Firebase Authentication.
تجربة المستخدم الجديدة :شاشات تعريفية تفاعلية للمستخدمين الجدد يتم التحكم بظهورها عبر SharedPreferences.
و جلب بيانات حية من REST API باستخدام مكتبة Volley وعرض الصور باستخدام Glide.
إدارة المواعيد: حجز المواعيد، تصفيتها (قادمة/ملغاة)، وإدارتها بشكل كامل.
نظام التنبيهات: تنبيهات محلية مخزنة في قاعدة بيانات SQLite لتتبع سجل الحجوزات.
المفضلة: حفظ النصائح الصحية المفضلة للوصول إليها لاحقاً بدون إنترنت.
واجهة مستخدم عصرية: تم بناء الواجهات باستخدام RecyclerView و BottomNavigationView وعناصر Material 3
